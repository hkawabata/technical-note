# tqdm 概要

for 文の実行時にプログレスバーを表示。
名称は progress に相当するアラビア語である taqadum に由来。

# インストール

```bash
$ pip install tqdm
```

# 使い方

```python
import tqdm
import time

for i in tqdm.tqdm(range(100)):
	time.sleep(0.05)  # プログレスバーが伸びていく様子を観察しやすいように sleep

"""

"""
```
